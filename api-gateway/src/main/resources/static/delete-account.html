계정 삭제 안내

스토맵은 유니멀 아틀라스가 운영하는 지도 위에 나만의 이야기를 핀으로 남기고 주변 사람들과 공유하는 위치 기반 스토리 공유 서비스입니다.
아래 안내에 따라 계정을 삭제할 수 있습니다.

계정 삭제 방법

앱 내에서 직접 계정을 삭제할 수 있습니다.

  1. 스토맵 앱을 실행합니다.
  2. 하단 탭에서 프로필 탭을 선택합니다.
  3. 내 개인정보를 선택합니다.
  4. 하단의 탈퇴하기를 선택합니다.
  5. 확인 창에서 탈퇴를 눌러 계정 삭제를 완료합니다.

삭제되는 데이터

계정 삭제 요청 즉시 아래 개인정보가 삭제(초기화)됩니다.

  • 이름, 닉네임, 프로필 사진
  • 휴대전화번호
  • 기기 정보 및 FCM 토큰

데이터 보관 기간

계정 삭제 요청 시 개인정보는 즉시 삭제되며, 이메일 및 활동 데이터(게시글, 댓글, 좋아요 등)는 삭제일로부터 6개월간 보관 후 완전 삭제됩니다.

단, 관계 법령에 따라 일부 기록은 아래 기간 동안 보관 후 파기됩니다.

  • 전자상거래 관련 기록: 최대 5년
  • 웹사이트 로그 기록: 3개월
  • 개인위치정보 관련 기록: 6개월

재가입 안내

계정 삭제 후 6개월 이내에는 아래 이메일로 재가입을 문의하실 수 있습니다.

  • 이메일: support@unimal.co.kr

6개월이 경과한 경우 계정 및 모든 데이터가 완전히 삭제되어 복구가 불가능합니다.

⚠️ 6개월 경과 후에는 계정을 복구할 수 없습니다. 삭제 전에 필요한 정보를 미리 저장해 주세요.
